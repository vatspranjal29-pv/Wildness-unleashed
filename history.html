How to start over without leaving anything behind?
But morning comes anyway.
It always does.

The world wakes up
like nothing inside me stayed awake all night.

And if my thoughts really had a search history,
I’d probably clear it.

Delete everything. Start fresh.
Pretend none of it happened.
But maybe that’s the problem.

Maybe it’s not meant to be erased.

Maybe it’s meant to be read—
slowly, carefully,
without skipping the uncomfortable parts.

Because somewhere between
“Why am I like this”
and
“How do I change”

there’s a version of me
still waiting to be understood.
